# Expanding Images

A simple and responsive Expanding Cards UI built using HTML, CSS, and JavaScript.

This project displays a set of image panels where clicking on a panel expands it while collapsing the others, creating a smooth and interactive visual effect.

Live Demo:
https://bhavanish-mantri.github.io/Expanding_images/

Features:
- Click-to-expand image panels
- Smooth CSS transitions
- Responsive layout for mobile devices
- Clean and minimal UI

Technologies Used:
- HTML
- CSS (Flexbox and transitions)
- JavaScript (DOM manipulation)
